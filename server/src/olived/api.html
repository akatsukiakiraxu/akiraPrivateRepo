/ml/settings/get get

機械学習の設定を取得する。

get

get /ml/settings/get

  • Response

HTTP status code 200

Body

Media type: application/json

Type: object

Properties
  • target_channels: (array of string)

    入力元のチャンネル。現在は同時に1つのチャンネルのみ使用可能。

  • input_data_offset: (integer)

    入力データの先頭からのオフセット

  • input_data_size: (integer)

    入力のサイズ

  • training_count: (integer)

    トレーニング回数

  • trained_network_path: (string)

    トレーニング済みネットワークのパス。指定しない場合はトレーニング

  • input_type: (one of raw, fft)

    入力の種類

  • loss_threshold: (number)

    ロス値の閾値

/ml/event get

イベントデータを通知するためのWebSocketエンドポイント。実際はWSなのでGETでアクセスはできない。

get

get /ml/event

  • Response

HTTP status code 200

Body

Media type: application/json

Possible types:

  • MLTrainingEvent

    • type: required (string - pattern: training-progress|training-done)

      ML関連イベントの種類

    • trained_count: required (number)

      トレーニング完了した入力データ数

  • MLInferenceEvent

    • type: required (string - pattern: inference-result)

      ML関連イベントの種類

    • loss: required (number)

      ロス値

/acquisition/config/get get

センサデータの収集ユニットの現在の構成を取得する。

get

get /acquisition/config/get

  • Response

HTTP status code 200

Body

Media type: application/json

Type: object

Properties
  • name: required (string)

    ユニットの名前

  • type: required (string)

    ユニットの種類

  • change_finder_list: required (array of string)

    Change Finderの候補リスト。現在のところ常に空のリストとなる。

  • channels: required (object)

    ユニットのチャンネル一覧。実際はキーをチャンネル名とする連想配列(JSではobject)

    • //: required (object)

      チャンネルの構成

      • sampling_rates_supported: required (array of number)

        サポートしているサンプリングレート

      • gains_supported: required (array of number)

        サポートしているゲイン

      • couplings_supported: required (array of AcquisitionCouplingType)

        サポートしているカップリング

      • minimum_voltage: required (number)

        最小入力電圧

      • maximum_voltage: required (number)

        最大入力電圧

      • minimum_value: required (number)

        ADCの最小値

      • maximum_value: required (number)

        ADCの最大値

Body

Properties

      チャンネルchNの設定。

        チャンネルの取り込みが有効かどうか

      • sampling_rate: (number)

        チャンネルのサンプリングレート

      • gain: (number)

        チャンネルのゲイン

      • coupling: (string - pattern: DC|AC)

        チャンネルのカップリング
